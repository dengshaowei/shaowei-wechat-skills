# 少威公众号图注样式版本表

## Caption V1｜普通简短图注
- 图下方一行
- 灰色小字
- 居中
- 最轻量

示例：
- 图 1｜AI 成立的三要素总览

## Caption V2｜明显居中图注
- 比 V1 更明显
- 字重更清晰
- 上下间距更统一
- 强化“这是图注”这件事

## Caption V3｜居中胶囊图注
- 胶囊底
- 更贴近图片下沿
- 视觉中心更强
- 适合公众号图文成品感更强的版本

## Caption V4｜视觉右移补偿图注
- 在 V3 基础上继续补偿
- 允许整体向右偏移
- 允许每张图单独微调
- 以微信渲染实际观感为准

## 当前建议
- 常规文章：V1 / V2
- 需要更强图文设计感：V3
- 用户明确反馈“不够居中”：V4
