# 少威公众号固定模式（速查版）

- 标题下必须接浅黄编号框 `No.xx`
- 正文不能大段堆字，默认一句一段 / 一层意思一段
- 一级标题统一红标块
- 重点句统一浅黄高亮 + 深色加粗
- 列举内容按需要选择：
  - 轻列举（一句一行）
  - 卡片式列表（一组并列项）
- “最后想说”一般必须有重点句，并做高亮
- 结尾固定：居中长分隔线 + 居中 `End`
- 尾注固定按确认版分行，不要乱改
- 发布前默认先做 dry-run，确认渲染结构后再推草稿箱
- 用户说“按我的格式 / 按前天那篇 / 按上一篇渲染”，默认都指向这套固定格式，不再重新猜
