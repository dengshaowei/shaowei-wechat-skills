# Shaowei Wechat Illustrator

## Role

这个 skill 是少威公众号体系里的**配图规划层**。

它负责：
- 判断文章是否需要配图
- 判断哪些位置适合插图
- 规划每张图的用途和风格
- 给出图注 / 图片说明建议
- 协调图文节奏，让版面更像成品

它不负责：
- 重写正文主体
- 微信接口发布
- 强绑定某一个图片平台或绘图模型

## Position In The Skill Suite

默认串联位置：

1. `shaowei-wechat-article`：内容生成
2. `shaowei-wechat-writer`：固定格式渲染
3. `shaowei-wechat-illustrator`：配图规划
4. `wechat-article-publisher`：预览 / 发布
5. `shaowei-wechat-pipeline`：总控编排

## Input

输入应尽量是：
- 已经写顺的文章正文
- 最好是已经完成主要格式渲染的定稿

## Output Contract

输出应包含以下内容中的全部或部分：

### 1. 是否建议配图
直接判断：
- 不需要配图
- 适合 1 张头图
- 适合 2-4 张文中配图
- 适合头图 + 文中配图组合

### 2. 配图位置规划
对文章进行分段标注，指出：
- 哪一段前后适合插图
- 为什么放在这里
- 这一张图承担什么作用

常见作用：
- 开场建立氛围
- 解释抽象概念
- 承接段落转场
- 提升阅读节奏
- 强化结尾情绪或落点

### 3. 图片类型建议
根据内容推荐图片类型，例如：
- 场景感插图
- 概念示意图
- 流程图 / 结构图
- 产品截图
- 数据图表
- 实拍图
- 封面图

### 4. 图片风格建议
给出轻量风格建议，例如：
- 简洁留白
- 科技感
- 温暖生活感
- 真实纪实感
- 手绘感
- 低饱和 / 高质感

### 5. 图注建议
若适合加图注，补一句简短说明。

图注要求：
- 简短
- 不抢正文
- 只起辅助理解或情绪承接作用

## Decision Rules

### 适合加图的情况
- 文章结构较长，需要节奏缓冲
- 有抽象概念，需要视觉解释
- 有明显场景切换
- 有产品、系统、机器人、界面等可视化对象
- 用户明确要求更像公众号成品

### 不适合乱加图的情况
- 文章很短
- 内容非常集中，插图会打断节奏
- 图片无法提供额外信息，只会干扰阅读

## Non-Negotiables

- 不要为了“看起来丰富”而乱加图
- 不要每个小节都硬插图
- 不要让图片抢走正文重点
- 不要让图注变成另一段正文解释

## Portability

这个 skill 默认工具无关：
- 可以只输出配图规划建议
- 可以接图库、设计工具、AI 生图工具或人工选图流程
- 如果未来要扩展成自动生成配图，只需要在这个 skill 基础上继续细化
